# Smart-Campus-Issue-Reporter-AI
The Smart Campus Issue Reporter is a real-time web platform designed to bridge the gap between students and campus administration. By leveraging Google Gemini AI, the system provides instant technical troubleshooting strategies for reported infrastructure issues, ensuring faster resolution and smarter maintenance.
✨ Key Features
Real-time Reporting: Students can submit maintenance requests with images, which are instantly synced to the cloud via Firebase.

AI Triage & Strategy: Integrates Gemini 1.5 Flash to analyze issue descriptions and generate 3-step technical fix recommendations for administrators.

Admin Dashboard: A secure management portal where staff can view evidence, track issue priority, and mark complaints as resolved.

Cross-Platform Sync: Real-time listeners ensure that reports submitted on mobile appear instantly on the admin laptop dashboard.

Cloud-Native Evidence: Supports image uploads converted to Base64 strings for seamless storage in the Firebase Realtime Database.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Custom Glassmorphism UI), Bootstrap 5.3

Backend: Firebase Realtime Database (Compat Version)

AI Engine: Google Gemini API (1.5 Flash / 2.0 Flash)

Icons: FontAwesome 6.4.0

Deployment: Firebase Hosting
